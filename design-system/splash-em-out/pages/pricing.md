# Pricing Page Override

Use this file for `/pricing`.

## Goal

Make price, minimum, turnaround, and service choice visible before persuasion.

## Required Content

- Drop-off wash-and-fold: price per pound, minimum, pickup timing.
- Pickup and delivery: recurring price, as-needed price, minimum, turnaround.
- Bulky item pricing when available.
- Promotions only if active and verified.
- Commercial quote explanation, not a fake fixed price.

## Design Rules

- Do not use SaaS tiers like Free, Pro, Enterprise.
- Use service-based pricing blocks with minimum and next action.
- Keep FAQs close to pricing objections.
- Use tabular figures for prices and phone numbers.
- Each pricing block needs an action: Schedule Pickup, Find a Location, or Request a Bid.
