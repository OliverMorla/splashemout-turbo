# Locations Page Override

Use this file for `/locations` and location detail pages.

## Goal

Help a mobile visitor choose a laundromat, call, or get directions quickly.

## Hub Requirements

- Map/list relationship should be clear.
- Every location preview needs address, phone, services, and directions.
- Use city grouping or search/filter when the list grows.
- Avoid hiding phone and directions behind hover.

## Detail Page Requirements

Each location page should include:

- Address.
- Phone.
- Hours or verification state.
- Services available.
- Machine capacity or amenities where known.
- Parking/access notes when known.
- Map embed or directions link.
- Nearby context only when useful and specific.

## Accessibility

Maps need a text/list alternative. Direction links and phone links must be keyboard reachable and screen-reader clear.
